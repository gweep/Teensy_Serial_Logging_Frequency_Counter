My version of the enclosure for the EtherKit EtherKeyer Mini.  

![image](https://github.com/gweep/EtherKeyerMini_AB2TS/blob/main/images/IMG_4878.HEIC)

Made changes to the original FreeCAD model to accomodate a two color enclosure.  Made subtle changes to the pc board mounting features and added a small and marginally effective miniMegaphone for the EtherKeyer Mini's piezo buzzer.
